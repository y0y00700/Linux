2장. zsh vs bash

sudo apt-get install zsh

zsh 가 설치되었고, zsh 을 통해 명령어를 해석 시킬수 있다.

bash와 zsh 은 부모가 같기 때문에 상당히 유사함.
zsh 는 좀더 편리한 기능을 가지고 있다.

zsh이 설치된 채로 
zsh + Enter 
시 zsh 모드로 변경된다.
bash + Enter 시, bash모드로 변경

echo $0 -> 현재 어떤 shell 을 사용중인지 출력
